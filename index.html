自己紹介

　私は過去3年間、建設コンサルタント業界で働き、主に道路や橋の設計・計画に携わってきました。社会的に重要な持続可能な交通インフラの構築に貢献し、道路や橋の設計、維持管理計画の策定、報告書やCAD図面の作成などを担当してきました。また、業務効率化のためにVBAによるマクロの作成を経験する中で、情報技術に深い興味を抱くようになりました。現在は、ゲームの制作やWebサイト制作にも取り組み、現在はスキルを向上させるために学習中です。

制作物

マッスルサンタ

　Unity1週間ゲームジャムに参加し、1週間という限られた時間内で開発からリリースまで行いました。このゲームは筋肉ムキムキのサンタがプレゼントを家に投げ入れるゲームです。楽しさランキングで334作品中37位にランクインしました。

使用した技術やツール

  • 　・C#
  • 　・Unity

ロボパズルシューター

　ロボパズルシューターは、多様なギミックとパズル要素を組み合わせた2Dアクションパズルゲームです。プレイヤーは、物体を大きくしたり小さくしたり、移動させながら、様々な障害物やパズルを解決してゴールを目指します。

使用した技術やツール

  • 　・C#
  • 　・Unity

Line Bot（サトー）

　このLine Botは、ユーザーの近くにいる人に通知をさりげなく見せるBotです。ユーザーが直接言うのが難しい場合に、間接的に情報を伝えることができます。 例えば飲み会の場合、早く帰る必要があることを周囲の人に知らせたい時などに便利です。

使用した技術やツール

  • 　・Node.js
  • 　・LINE Messaging API
  • 　・デプロイ: Render上でホスティング

網走周辺観光シュミレーター

　網走周辺を観光する際に、出発地と目的地を選んでランダムにルートを生成するウェブサイトです。網走を訪れた旅行者が知床や阿寒へ行く際に、おすすめのスポットを経由するルートを提案します。今後、出発地と目的地を追加する可能性があります。

使用した技術やツール

  • 　・HTML, CSS, JavaScript
  • 　・Node.js
  • 　・Google Maps API
  • 　・デプロイ: Render上でホスティング
